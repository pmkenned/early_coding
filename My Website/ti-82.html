Programming with the TI-82

   At my high school I am taking Algebra 2 and I have been issued a TI-82 graphing calculator. The TI-82 allows the user to create graphs and to write and store programs. Here you can learn how to program with the TI-82 and use a few programs I have written.


Creating, Editing, and Executing Programs

To create a program, press the [PRGM] button, move to NEW, and type the name of your program. The name can be up to 8 characters long, can contain the characters 0-9 A-Z, and it must begin with a letter.
To edit a program, press the [PRGM] button, move to EDIT, and select the program you want to edit.
To execute a program, press the [PRGM] button and under EXEC select the program you want to execute.

Controlling Program Flow

If Statements

   The If command can instruct a calculator to perform a specific operation if a certain condition if true. Example:

:If condition		// If Command and If Condition Line
:command it true	// The program only executes this command if the condition is true and skips if false
:command		// After the If command is complete the program continues with the other commands

:If x=1			// If the variable x equals one
:a+10[STO]a		// Add 10 to the variable a and store it as a
:10a[STO]a		// Multiply 10 by a and store it as a
If-Then Statements

   A If-Then statement allows for multiple lines of commands to be performed if the condition is true. Example:

:If condition		// If Command and If Condition Line
:Then			// Begins the command group for a true outcome
:command it true	// The program only executes this command if the condition is true and skips if false
:command it true	// The program only executes this command if the condition is true and skips if false
:End			// Closes the the If-Then group
:command		// After the If command is complete the program continues with the other commands

:If x=1			// If the variable x equals one
:Then			// Begins the command group for a true outcome
:a+10[STO]a		// Add 10 to the variable a and store it as a
:b+1[STO]b		// Add 1 to the variable b and store it as b
:End			// Closes the If-Then group
:10a[STO]a		// Multiply 10 by a and store it as a
If-Then-Else Statements

   A If-Then-Else statement allows for multiple lines of commands to be performed if the condition is true and multiple lines for a false outcome. Example:

:If condition		// If Command and If Condition Line
:Then			// Begins the command group for a true outcome
:command it true	// The program only executes this command if the condition is true and skips if false
:command it true	// The program only executes this command if the condition is true and skips if false
:Else			// Begins the command group for a false outcome
:command if false	// The program only executes this command if the condition is false
:command if false	// The program only executes this command if the condition is false
:End			// Closes the If-Then-Else group
:command		// After the If command is complete the program continues with the other commands

:If x=1			// If the variable x equals one
:Then			// Begins the command group for a true outcome
:a+10[STO]a		// Add 10 to the variable a and store it as a
:b+1[STO]b		// Add 1 to the variable b and store it as b
:Else			// Begins the command group for a false outcome
:a-10[STO]a		// Subtracts 10 from the variable a and store it as a
:b-1[STO]b		// Subtracts 1 from the variable b and store it as b
:End			// Closes the If-Then-Else group
:10a[STO]a		// Multiply 10 by a and store it as a
While Statements

   The While command will repeat a certain operation until a condition is no longer true. Example:

:While condition	// While Command and While Condition Line
:command		// Repeats command until condition is false
:command		// Repeats command until condition is false
:End			// Closes the repeat command group
:command		// After the While command is complete the program continues with the other commands

:While x<100		// Repeat until x is not less than 100
:a+1[STO]a		// Add 10 to the variable a and store it as a
:x+2[STO]x		// Add 1 to the variable b and store it as b
:End			// Closes While command group
:10a[STO]a		// Multiply 10 by a and store it as a
Repeat Statements

   The Repeat command will repeat a certain operation until a condition has been met. Example:

:Repeat condition	// Repeat Command and Repeat Condition line
:command		// Repeats command until condition is true
:command		// Repeats command until condition is true
:End			// Closes the repeat command group
:command		// After the If command is complete the program continues with the other commands

:Repeat x=100		// Repeat until x equals 100
:a+1[STO]a		// Add 10 to the variable a and store it as a
:x+2[STO]x		// Add 1 to the variable b and store it as b
:End			// Closes the If-Then-Else group
:10a[STO]a		// Multiply 10 by a and store it as a
Tests and Logic

By using tests you can control program flow with control statements. The kinds of tests are equal to, not equal to, greater than, greater than or equal to, less than, and less than or equal to. Examples:

If x=1
If x>1
If x<1

While controlling program flow you may want to perform commands if one of two conditions is true, only if both are true, only if one is true and the other false etc. You can do this using logic. The logic commands are and, or, xor (exclusive or), and not. Examples:

If x=1 and y=2	// The condition is only true if both parts are true
If x=1 or x=2	// The condition is true if either parts are true
If x=1 xor x=2	// The condition is true only if one part is true and the other is false
If not x	// The condition is true if the value is 0
